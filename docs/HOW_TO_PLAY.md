## For Students

### Joining a Game

1. **Get the Game Code**
   - Your teacher will share a 6-character game code (e.g., `ABC123`)
   - This code is unique to your classroom's game session

2. **Join the Game**
   - Navigate to **"Join Game"** from the sidebar
   - Enter the 6-character game code
   - Enter your display name (or use your account name)
   - Click **"Join Game"**

3. **Wait in the Lobby**
   - You'll enter a lobby showing all connected players
   - Wait for your teacher to start the game
   - The lobby displays:
     - Number of players connected
     - Player names
     - Game settings (question set, time limits, etc.)
   - **Navigation:** Top-left button shows "Return to Dashboard" (game hasn't started yet)

4. **Play the Game**
   - Questions appear one at a time
   - Read the question and select your answer
   - You have a time limit to answer (shown by a countdown timer)
   - Answer feedback:
     - ‚úÖ **Correct** - Green highlight, points awarded, business driver badge shown
     - ‚ùå **Incorrect** - Red highlight, shows correct answer
   - **Business Driver Badges:** Each question displays which of the 6 P's it represents:
     - üë• People, üì¶ Product, üí∞ Pricing, ‚öôÔ∏è Process, üìà Proceeds, üíé Profits
   - Wait for the next question (teacher controls progression)

5. **View Results**
   - After all questions are complete, see:
     - Your final score
     - Your placement/rank
     - Pathkeys earned
     - Tokens (XP) earned
   - Click **"Return to Dashboard"** when done

### Collecting Pathkeys

- **What are Pathkeys?** Digital awards representing mastery in career exploration
- **Three Progressive Sections:** Each pathkey has three unlock sections:
  - üéØ **Career Image** - Finish Top 3 in Career mode
  - üîí **Lock** - Complete 3 Industry or Cluster question sets with 90% accuracy
  - üîë **Key** - Master all 6 business drivers (5 questions each at 90% accuracy)
- **How to Earn:** See detailed requirements in [Pathkey Award System](#pathkey-award-system)
- **Where to View:** Navigate to **"My Pathkeys"** to see your collection and progress
- **Track Progress:** Each pathkey shows which sections you've unlocked

## For Teachers - Standalone Classroom Games

### Creating a Classroom Game

1. **Navigate to Host Game**
   - Click **"Host Game"** from the sidebar
   - You'll see the game setup page

2. **Configure Your Game**

   **Step 1: Filter Question Sets**

   *Exploration Type* (Required)
   - **Industry** - Broad industry overviews (e.g., Healthcare, Technology)
   - **Career** - Specific careers (e.g., Software Engineer, 3D Artist)
   - **Cluster** - CTE Career Cluster fundamentals (e.g., STEM, Business)

   *Difficulty Level* (Optional)
   - **Easy** - Introductory questions
   - **Medium** - Standard difficulty
   - **Hard** - Advanced questions
   - **Mixed** - Variety of difficulty levels

   **Step 2: Select Question Set**
   - Choose from 191 available question sets with 3,000+ questions
   - Each set has 12-60 questions depending on type

   **Step 3: Choose Question Count**
   - Select how many questions: **10, 15, 20, 25, or 30**
   - Questions are randomly selected from the full pool
   - Ensures fresh content even when replaying the same set

   **Step 4: Game Settings**

   *Session Type*
   - **Multiplayer** - Multiple students in competitive game
   - **Solo Practice** - Test questions yourself before using with students

   *Game Mode*
   - **Career Quest** - Classic quiz mode with career-themed questions
   - **Lightning! Path** - Fast-paced questions with time pressure

   *Progression Control*
   - **Manual** - You click "Next Question" to advance
   - **Auto** - Questions auto-advance when timer expires

   *Optional Settings*
   - **Max Players** - Limit on number of students (default: 30)
   - **Allow Late Join** - Let students join after game starts
   - **Public/Private** - Make game discoverable or code-only

3. **Start the Game**
   - Click **"Create Game"**
   - A unique 6-character **Game Code** is generated (e.g., `XYZ789`)
   - You'll be taken to the **Game Lobby**

4. **Share the Game Code**
   - Display the game code on your screen
   - Students enter this code to join
   - Watch as students connect in real-time

5. **Monitor the Lobby**
   - See all connected players
   - View player names and connection status
   - Wait until all students have joined
   - **Navigation Tip:** Top-left button says "Return to Dashboard" before game starts, "Leave Game" after

6. **Start Playing**
   - Click **"Start Game"** when ready (standalone games)
   - Or wait for tournament coordinator to start (coordinated tournaments)
   - Questions begin appearing for students
   - You'll see the **Host View**:
     - Current question with correct answer highlighted
     - Real-time player responses
     - Live leaderboard
     - Timer countdown

7. **Control Game Flow**

   **Manual Mode:**
   - Review player responses
   - Click **"Next Question"** when ready to advance
   - You control the pace

   **Auto Mode:**
   - Questions auto-advance when timer expires
   - You monitor but don't click next
   - Consistent pacing for all students

8. **End Game**
   - After final question, click **"End Game"**
   - Students see their results
   - You see final leaderboard with:
     - Player rankings
     - Scores
     - Accuracy percentages
     - Rewards distributed (Pathkeys, Tokens)

9. **Post-Game**
   - Review results on **Analytics** page
   - Track student progress over time
   - Export data if needed

## For Teachers - Tournament Mode

### Understanding Tournaments

**What is a Tournament?**
- Multi-classroom competition
- Multiple teachers host their own classroom games
- All classrooms compete in the same tournament
- Aggregate leaderboards show top players across all classrooms
- Classroom rankings show which classroom performed best

**Tournament Roles:**
- **Tournament Coordinator** - Creates and manages the tournament
- **Classroom Host** - Joins tournament with their classroom

### Creating a Tournament (Coordinator)

1. **Navigate to Create Tournament**
   - Click **"Create Tournament"** from sidebar
   - Fill out tournament details

2. **Configure Tournament**

   **Required Information:**
   - **Title** - Tournament name (e.g., "Career Takeover Challenge")
   - **Description** - Brief description of the tournament
   - **Question Set** - Select which question set to use
   - **School Name** - (Optional) Your school name

   **Tournament Settings:**

   **Start Mode:**
   - **Coordinated** - Coordinator starts all classrooms simultaneously
   - **Independent** - Each classroom starts when their teacher is ready

   **Progression Mode:**
   - **Manual** - Teachers control question advancement
   - **Auto** - Questions auto-advance based on timer

   **Limits:**
   - **Max Classrooms** - Maximum number of classrooms (default: 20)
   - **Max Players per Classroom** - Maximum students per class (default: 60)
   - **Allow Late Join** - Whether students can join after start

3. **Create Tournament**
   - Click **"Create Tournament"**
   - A unique 6-character **Tournament Code** is generated (e.g., `ABC123`)
   - Tournament status: **"Ready"**
   - You're taken to the **Tournament Coordinator Page**

4. **Share Tournament Code**
   - Share the tournament code with other teachers
   - They will use this code to join with their classrooms
   - The tournament code is displayed prominently on the coordinator page

5. **Monitor Tournament Setup**

   **Tournament Coordinator Dashboard Shows:**
   - **Tournament Code Card** (purple gradient) - Share this with teachers
   - Tournament status badge (Ready, Waiting, In Progress, Completed)
   - Number of classrooms joined
   - Total players across all classrooms
   - List of joined classrooms with their game codes
   - Start mode and progression settings
   - Real-time leaderboards and rankings

   **Coordinator vs Participant Views:**
   - **Coordinators** see:
     - Tournament code (to share with teachers)
     - "Start Tournament" button (coordinated mode)
     - Full control over tournament
   - **Participants** (teachers who joined) see:
     - "My Classroom" card with their game code
     - "Go to Game Lobby" button
     - Tournament leaderboards and rankings
     - No tournament code or start button

   **Status Updates:**
   - **Ready** - Tournament created, waiting for classrooms to join
   - **Waiting** - At least one classroom joined, ready to start
   - **In Progress** - Tournament is active
   - **Completed** - Tournament finished, view results

6. **Start Tournament (Coordinated Mode)**
   - Wait for classrooms to join
   - Click **"Start Tournament"** when ready
   - All classroom games start simultaneously
   - Status changes to **"In Progress"**

7. **Monitor During Tournament**

   **Real-time Data:**
   - **Tournament Leaderboard** - Top 50 players across all classrooms
   - **Classroom Rankings** - Which classroom is performing best
   - **Live Updates** - See scores update as students answer

   **View Options:**
   - Switch between tabs:
     - Classrooms (list of all classrooms)
     - Tournament Leaderboard (individual players)
     - Classroom Rankings (team competition)

8. **Tournament Completion**
   - Games end when all questions are answered
   - Final results calculated:
     - Individual player rankings (tournament-wide)
     - Classroom team rankings (total/average scores)
   - Status changes to **"Completed"**
   - Results remain accessible for review

### Joining a Tournament (Classroom Host)

1. **Get Tournament Code**
   - Receive the 6-character tournament code from the coordinator
   - Example: `XYZ789`

2. **Navigate to Join Tournament**
   - Click **"Join Tournament"** from sidebar
   - Enter the tournament code
   - Click **"Find Tournament"**

3. **Review Tournament Details**
   - Tournament title and description
   - Start mode (Coordinated or Independent)
   - Progression mode (Manual or Auto)
   - Coordinator's school name

4. **Enter Classroom Name**
   - Provide a unique name for your classroom
   - Examples: "Mr. Smith's Math Class", "Room 205", "Period 3"
   - This helps students identify their classroom in leaderboards
   - Click **"Join Tournament"**

5. **Receive Student Game Code**
   - A unique 6-character **Game Code** is generated for your classroom
   - Example: `DEF456`
   - This is DIFFERENT from the tournament code
   - Students use THIS code to join YOUR classroom

6. **Share Student Game Code**
   - Display the game code for your students
   - Students join using the normal "Join Game" flow
   - They'll be joining your classroom within the tournament

7. **Wait in Lobby**
   - Students join your classroom game
   - You see them in the lobby
   - Wait for tournament coordinator to start (Coordinated mode)
   - OR start when ready (Independent mode)

8. **Host Your Classroom Game**
   - The game plays like a normal classroom game
   - You use the **Host View** to monitor your students
   - Control progression (if manual mode)
   - Your classroom competes against other classrooms

9. **Access Tournament Dashboard**
   - Click on the tournament card in "My Tournaments" on your dashboard
   - You'll see a **Participant View** showing:
     - **My Classroom** card (teal/cyan) with:
       - Your classroom name
       - Student game code (for your students to join)
       - "Go to Game Lobby" button
     - Tournament title and status
     - Tournament leaderboard (all students across classrooms)
     - Classroom rankings (see how your class compares)
   - Note: You won't see the tournament code (coordinator only) or "Start Tournament" button

10. **Manage Your Classroom Game**
   - From the tournament dashboard, click **"Go to Game Lobby"**
   - This takes you to your classroom's game lobby
   - Share your unique game code with students
   - Students join YOUR classroom using this code
   - Wait for tournament to start (coordinated mode) or start when ready (independent mode)

11. **View Tournament Results**
   - Return to tournament dashboard at any time
   - See real-time updates:
     - Your classroom's ranking
     - Top students in your classroom
     - Tournament-wide leaderboard
     - Other classroom rankings

## Game Modes & Settings

### Session Types

**Solo Practice**
- Single player practicing alone
- Self-paced (student clicks "Next Question")
- Ideal for homework or independent study
- Earns Pathkeys and Tokens

**Multiplayer**
- Multiple students in same game session
- Teacher controls as host
- Competitive or collaborative
- Live leaderboard

### Progression Control

**Manual Progression**
- **Teacher Control** - Teacher clicks "Next Question"
- **Benefits:**
  - Control pacing
  - Time for discussion
  - Explain answers between questions
  - Wait for all students
- **Best For:**
  - In-class instruction
  - Discussion-based learning
  - Mixed-ability groups

**Auto Progression**
- **Timer Control** - Questions auto-advance when timer expires
- **Benefits:**
  - Consistent pacing
  - Timed challenge
  - No teacher intervention needed
  - Simulates test conditions
- **Best For:**
  - Assessments
  - Fast-paced competitions
  - Self-directed learning

### Game Modes

**Career Quest**
- Classic quiz mode with career-themed questions
- Questions cover all aspects of career exploration
- All questions tagged with business drivers (6 P's framework)
- Suitable for all students

**Lightning! Path**
- Fast-paced questions with time pressure
- Shorter time limits create exciting competition
- Same career content, faster gameplay
- Great for energizing the classroom

**Business Driver Framework (6 P's)**
All questions are tagged with one of the six business drivers:
- **People** - Team management, collaboration, communication
- **Product** - Services, innovation, quality, tools
- **Pricing** - Budgeting, financial planning, compensation
- **Process** - Workflows, procedures, compliance
- **Proceeds** - Career growth, opportunities, networking
- **Profits** - Ethics, strategy, sustainability

### Tournament Modes

**Coordinated Start**
- All classrooms start at the same time
- Ensures fair competition
- Coordinator controls when tournament begins
- Best for: Scheduled events, assemblies, school-wide competitions

**Independent Start**
- Each classroom starts when ready
- Flexible timing for different class periods
- Teachers start their own classroom games
- Best for: Different schedules, time zones, multi-day events

## Scoring & Rewards

### How Scoring Works

**Base Score Formula:**
```
Score = Correct Answers √ó Base Points √ó Speed Multiplier
```

**Points Breakdown:**
- **Correct Answer** - Base points awarded
- **Speed Bonus** - Faster answers = higher multiplier
- **Streak Bonus** - Consecutive correct answers (if enabled)
- **Difficulty Bonus** - Harder questions worth more (if applicable)

**Example:**
- Question worth 100 points
- Answered in 3 seconds (80% of time remaining)
- Speed multiplier: 1.8√ó
- Final score: 100 √ó 1.8 = 180 points

### Leaderboard Rankings

**Individual Games:**
- Ranked by total score
- Tiebreaker: Faster total time
- Placement determines rewards

**Tournaments:**
- **Individual Leaderboard** - Top players across all classrooms
- **Classroom Rankings** - Based on classroom total/average scores
- Both leaderboards shown to coordinator and participants

### Pathkey Award System

**What are Pathkeys?**

Pathkeys are digital awards that represent mastery in career exploration. Each pathkey is tied to a specific career (like Public Relations Specialist, Software Engineer, Nurse, etc.) and has **three progressive sections** that unlock as you demonstrate different types of mastery.

**Three Progressive Sections**

Each pathkey has three unlock sections that must be earned in order:

**Section 1: Career Mastery (Career Image) üéØ**
- **Requirement:** Finish in the **Top 3** in a Career mode game for that career
- **Unlocks:** The career image showing the professional in action
- **Competition:** Requires 3+ total players for competitive integrity
- **Example:** Play "Public Relations Specialist" career mode, finish 1st, 2nd, or 3rd place

**Section 2: Industry/Cluster Mastery (Lock) üîí**
- **Prerequisite:** Must have Section 1 unlocked first
- **Requirement:** Complete **3 question sets** with **90% accuracy or better**
- **Two Paths (choose ONE):**
  - **Industry Path:** Play 3 Industry-level question sets matching the career's sector
    - Example: For PR Specialist (sector: "Marketing & Community"), play 3 Marketing & Community Industry sets
  - **Cluster Path:** Play 3 Cluster-level question sets matching the career's cluster
    - Example: For PR Specialist (cluster: "Marketing"), play 3 Marketing cluster sets
- **Unlocks:** The lock image
- **Progress:** Tracked automatically across all games

**Section 3: Business Driver Mastery (Key) üîë**
- **Prerequisite:** Must have Section 1 unlocked (Section 2 not required)
- **Requirement:** Master **all 6 business drivers** by answering questions correctly
- **How it Works:**
  - Questions are tracked in **chunks of 5**
  - Each chunk must achieve **90% accuracy** (4 or 5 correct out of 5)
  - If a chunk fails, it resets and you try again
  - Progress accumulates across multiple game sessions
- **Six Business Drivers (6 P's):**
  - üë• **People** - Team management, collaboration, communication
  - üì¶ **Product** - Services, innovation, quality, tools
  - üí∞ **Pricing** - Budgeting, financial planning, compensation
  - ‚öôÔ∏è **Process** - Workflows, procedures, compliance
  - üìà **Proceeds** - Career growth, opportunities, networking
  - üíé **Profits** - Ethics, strategy, sustainability
- **Unlocks:** The key image
- **Only in Career Mode:** Business driver tracking only works in Career mode games

**Example: Earning a Complete Pathkey for Public Relations Specialist**

1. **Play Career Mode** ‚Üí "Public Relations Specialist" question set ‚Üí Finish in Top 3 ‚Üí ‚úÖ Career image unlocked

2. **Choose Your Path for the Lock:**
   - **Option A (Industry):** Play "Marketing & Community" Industry sets, achieve 90%+ in 3 sets ‚Üí ‚úÖ Lock unlocked
   - **Option B (Cluster):** Play "Marketing" Cluster sets, achieve 90%+ in 3 sets ‚Üí ‚úÖ Lock unlocked

3. **Master Business Drivers:**
   - Play PR Specialist Career games
   - Answer 5 "People" questions with 90% accuracy (chunk complete!)
   - Answer 5 "Product" questions with 90% accuracy (chunk complete!)
   - Continue for all 6 drivers (People, Product, Pricing, Process, Proceeds, Profits)
   - All 6 complete ‚Üí ‚úÖ Key unlocked

**Complete Pathkey:** Career image + Lock + Key = Full mastery of Public Relations Specialist pathway!

**Viewing Your Pathkeys**
- Navigate to **"My Pathkeys"** from the sidebar
- See all careers you've unlocked (even partial)
- Track progress toward each section
- View which business drivers still need mastery
- See Industry/Cluster progress counters

**Tokens (Experience Points)**
- Earned for participation and performance in all games
- Base tokens: 10 per game + score/10
- Placement bonuses:
  - 1st place: +50 tokens
  - 2nd place: +30 tokens
  - 3rd place: +20 tokens
- Used for progression tracking
- Future: May unlock premium features
- Displayed on dashboard

## Troubleshooting & Tips

### For Students

**Can't Join Game:**
- Verify game code is correct (6 characters, no spaces)
- Check if late join is allowed
- Make sure game hasn't ended
- Ask teacher to verify you're using the right code

**Lost Connection:**
- Refresh the page to reconnect
- Game progress is saved
- You can rejoin with the same code

**Pathkey Questions:**
- **Didn't unlock Section 1:** Requires Top 3 finish with 3+ players
- **Section 2 not progressing:** Must unlock Section 1 first, then achieve 90%+ accuracy
- **Business driver chunk reset:** Accuracy fell below 90% in the chunk of 5
- **Can't see progress:** Navigate to "My Pathkeys" to view all progress
- **Only works for registered users:** Guest players don't earn pathkeys

### For Teachers

**Students Can't Join:**
- Verify you shared the correct game code (NOT tournament code)
- Check max players limit
- Ensure game is still in "Waiting" status
- Enable "Allow Late Join" if game started

**Tournament vs Game Codes:**
- **Tournament Code** - Shared with other teachers to join tournament
- **Game Code** - Given to students to join your specific classroom
- These are DIFFERENT codes!

**Dashboard Doesn't Show Tournament:**
- Refresh the page (data updates every 30 seconds)
- Check that tournament isn't cancelled (only cancelled tournaments are hidden)
- Completed tournaments remain visible for reviewing results
- Verify you're logged in with correct account

**Navigation Tips:**
- From game lobby: Click "Return to Dashboard" (before game starts)
- From active game: Click "Leave Game"
- From tournament dashboard: Click tournament title or classroom cards
- Always use buttons in the app (not browser back button) for best experience

**Participant Can't See Tournament:**
- Verify you joined the tournament successfully
- Check "My Tournaments" on dashboard - it should appear there
- Refresh if needed (data updates every 30 seconds)
- Click the tournament card to see your classroom and leaderboards

## Best Practices

### For Teachers - Standalone Games

1. **Pre-Game Setup:**
   - Test the game with 1-2 students first
   - Choose appropriate time limits for your class
   - Decide manual vs auto before creating
   - Have game code ready to display

2. **During Game:**
   - Monitor the host view for engagement
   - Use manual mode if you want to discuss answers
   - Watch the leaderboard for outliers (too fast = guessing)
   - Keep the energy high!

3. **Post-Game:**
   - Review results in Analytics
   - Celebrate top performers
   - Identify struggling students
   - Use data to inform instruction

4. **Pathkey Strategy:**
   - **Section 1 (Career Image):** Encourage Career mode games for competition
   - **Section 2 (Lock):** Mix Industry and Cluster games to give students options
   - **Section 3 (Key):** Multiple Career games needed for all 6 business drivers
   - **Explain the System:** Help students understand the progressive unlock structure
   - **Track Progress:** Remind students to check "My Pathkeys" to see their progress

### For Tournament Coordinators

1. **Planning:**
   - Schedule tournament start time
   - Communicate with participating teachers
   - Choose question set everyone can access
   - Decide coordinated vs independent start

2. **Setup:**
   - Create tournament 24 hours in advance
   - Share tournament code with teachers early
   - Set realistic max classrooms/players
   - Test with one classroom first

3. **During Tournament:**
   - Monitor the coordinator dashboard
   - Be available for teacher questions
   - Watch for technical issues
   - Celebrate milestones (first classroom, 100 players, etc.)

4. **Post-Tournament:**
   - Share results with all participants
   - Recognize top classroom and individual winners
   - Gather feedback for next tournament
   - Keep results accessible for teachers

### For Classroom Hosts in Tournaments

1. **Before Joining:**
   - Confirm tournament date/time with coordinator
   - Prepare students (explain tournament format)
   - Test your classroom setup
   - Choose clear classroom name

2. **Joining:**
   - Join tournament 15-30 minutes before start
   - Share student game code clearly
   - Get students in lobby early
   - Coordinate with tournament start time

3. **During Tournament:**
   - Encourage students but keep fair play
   - Monitor your host view
   - Follow progression mode (manual or auto)
   - Track your classroom's performance

4. **After Tournament:**
   - Review classroom performance
   - Check tournament-wide leaderboards
   - Celebrate student achievements
   - Discuss results with class
